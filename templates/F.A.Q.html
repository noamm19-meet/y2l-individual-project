Do I need to use a gym for Sportogress workouts?

Not at all. However, if you’d like more hands-on training, you can go to 1:1 session page.

What should I do if a specific week of the program is very challenging?

Don’t fret! You can always repeat the week you’re on if you feel it was too difficult to move past.

How do I know which element to start with?

Each element has a list of prerequisites that you should meet before attempting it. Don’t be afraid to start with the basics and work your way to the top. We all have to start somewhere, and sometimes we just need a refresher. Whichever level you start at, the exercises you do should help prepare you for the next level of difficulty!

I'm trying to register, but it’s not working?

Unfortunately, this happens sometimes – but don’t get discouraged! Make sure you have an active and stable Internet connection when trying to register. We recommend using a strong, secure Wi-Fi connection for a fast, effective registration.
